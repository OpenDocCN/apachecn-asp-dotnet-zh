## 下载

### Docker

```
docker pull apachecn0/flygon-asp-dotnet-zh
docker run -tid -p <port>:80 apachecn0/flygon-asp-dotnet-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-asp-dotnet-zh
flygon-asp-dotnet-zh <port>
# 访问 http://localhost:{port} 查看文档
```
